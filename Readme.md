# Hematite

Hematite is a simple Minecraft Server for 1.21.10 written in Rust.

It is currently very WIP, almost nothing exists. You can't even join yet.

## Documentation

The protocol for Minecraft: Java Edition is very well documented [on their wiki](https://minecraft.wiki/w/Java_Edition_protocol/Packets) and my main source of information.
